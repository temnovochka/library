# Тестовый проект

Реализован RESTful API для CRUD операций для работы со следующими сущностями:

1. Книга - обязательные атрибуты: название, язык, год публикации
2. Автор - обязательные атрибуты: фио
3. Подписчик - обязательные атрибуты: фио, электронный адрес

Для языков создана отдельная сущность, где обязательным атрибутом является название.

## Запуск проекта

```
git clone https://github.com/temnovochka/library.git
cd library
docker-compose up -d
```

## Тестирование

Проверка работоспособности API осуществлялась с помощью `pytest`. Запуск тестов:

```
cd library
pip install -r requirements.txt
pip install -r test_requirements.txt
pytest
```
